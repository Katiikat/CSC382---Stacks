Assignment: Stacks

Assignment Requirements: 
For this assignment, you are to code a stack OR a queue.  You can use a library to implement this.  You must have functionality for check for empty, add, remove, print, and update.

Guidelines

You are NOT to cut and paste code from anywhere.  You are to implement this manually or through a library of your choice.

Requirements

You must implement and demonstrate the functionality of all the functions in your code.
